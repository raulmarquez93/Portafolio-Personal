Chat Bot

This chatbot, built with Flask and Python, uses a CSV file to store frequently asked questions and their corresponding answers. It leverages fuzzywuzzy to identify similarities in user queries and provide accurate responses. Googletrans is integrated for translation of answers, and Flask-CORS ensures the chatbot is accessible from different domains. The system is designed to answer questions about the creator, enhancing interaction through an easy-to-use web interface.
